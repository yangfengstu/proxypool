### 🎯 关键特性

**代理详情包含：**
- 代理地址 → 真实出口IP 映射 ⭐
- 过期状态和剩余时间
- 完整使用统计
- 健康评分和指标
- 预检信息

**日志格式：**
```
proxy=192.168.1.1:1080 type=socks5 exit_ip=1.2.3.4 use=10 health=0.85
```

### 🚀 使用示例

```go
// 1. 获取所有代理详情
details := pool.GetAllProxyDetails()
for _, d := range details {
    fmt.Printf("%s -> %s
", d.ProxyHost, d.RealExitIP)
}

// 2. 导出JSON
jsonData, _ := pool.ExportProxyDetailsJSON()
ioutil.WriteFile("proxies.json", []byte(jsonData), 0644)

// 3. 日志打印
info := pool.GetProxyInfo(proxy)
log.Printf("Request started %s", info)
// 输出: Request started proxy=192.168.1.1:1080 type=socks5 exit_ip=1.2.3.4 use=10 health=0.85

// 4. 安全URL（隐藏密码）
safeURL := proxy.SafeURL()  // socks5://user:pa****23@host:1080
```
